# Deployment And Setup

## Target Environment

- OS: Windows (project paths and batch tooling are Windows-oriented).
- Python: 3.12 recommended.
- Runtime mode: local desktop app.

## Project Bootstrap

1. Open terminal in project root.
2. Create virtualenv:
   - `python -m venv terr`
3. Activate:
   - `terr\Scripts\activate`
4. Install dependencies:
   - `pip install -r requirements.txt`
5. Install Playwright browsers:
   - `playwright install chromium`

## Configuration File

Main config file:

- `data/config.env`

Minimum required variables:

- `CLAVE_LOGIN`
- `RUT`
- `CLAVE`
- `RUT2`
- `CLAVE2`
- `RUTA_PDF_DTE_RECIBIDOS`
- `DB_PATH_DTE_RECIBIDOS`
- `OPENAI_API_KEY`
- `AI_OPENAI_MODEL`

Classification tuning variables:

- `OPENAI_ASSISTANT_ID`
- `REVIEW_BATCH_SLEEP`
- `REVIEW_MAX_RETRIES_OPENAI`
- `REVIEW_TEMPERATURE`
- `REVIEW_LIMIT_ROWS`
- `ASSISTANT_TRIGGER_CONFIDENCE`
- `ASSISTANT_MAX_WAIT_SECONDS`
- `ASSISTANT_POLL_SECONDS`

Pipeline behavior flags:

- `AI_DETALLE_ENABLED`
- `AI_DETALLE_DEBUG`
- `AI_CATEGORIA_ENABLED`
- `GUI_AI_THROTTLE_SECONDS`
- `GUI_REVIEW_BATCH_SLEEP`

Background weekly checker:

- `AUTO_WEEKLY_DTE_CHECK_ENABLED`
- `AUTO_WEEKLY_DTE_CHECK_INTERVAL_DAYS`
- `AUTO_WEEKLY_DTE_CHECK_RETRY_HOURS`
- `AUTO_WEEKLY_DTE_CHECK_POLL_MINUTES`

## Example config.env Template

```env
CLAVE_LOGIN=CHANGE_ME
RUT=XXXXXXXX-X
CLAVE=CHANGE_ME
RUT2=XXXXXXXX-X
CLAVE2=CHANGE_ME

RUTA_PDF_DTE_RECIBIDOS=C:/Users/USER/Documents/TERRALIX/data/pdfs
DB_PATH_DTE_RECIBIDOS=C:/Users/USER/Documents/TERRALIX/data/DteRecibidos_db.db

OPENAI_API_KEY=sk-...
AI_OPENAI_MODEL=gpt-4.1-mini
OPENAI_ASSISTANT_ID=asst_...

AI_DETALLE_ENABLED=true
AI_DETALLE_DEBUG=false
AI_CATEGORIA_ENABLED=false

REVIEW_BATCH_SLEEP=0.25
REVIEW_MAX_RETRIES_OPENAI=8
REVIEW_TEMPERATURE=0
REVIEW_LIMIT_ROWS=0

ASSISTANT_TRIGGER_CONFIDENCE=75
ASSISTANT_MAX_WAIT_SECONDS=45
ASSISTANT_POLL_SECONDS=1.0

AUTO_WEEKLY_DTE_CHECK_ENABLED=true
AUTO_WEEKLY_DTE_CHECK_INTERVAL_DAYS=7
AUTO_WEEKLY_DTE_CHECK_RETRY_HOURS=6
AUTO_WEEKLY_DTE_CHECK_POLL_MINUTES=30
```

## Run Modes

GUI mode:

- `python TERRALIX.py`
- or `tools\run-terralix.bat`

Batch scripts:

- Read one/all PDFs with AI:
  - `python app/core/DTE_Recibidos/ai_reader.py --dir "<PDF_DIR>"`
- Re-run categorization:
  - `python app/core/DTE_Recibidos/categorizer.py`

## Build (PyInstaller)

Spec file:

- `TERRALIX.spec`

Build command:

- `pyinstaller TERRALIX.spec --noconfirm`

Output bundle:

- `dist/TERRALIX/` (when generated by build command)
